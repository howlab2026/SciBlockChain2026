# SciBlockChain 개발 완료 보고서 (Walkthrough)

과학제전과 같은 행사에서 관람객이 카드 결제를 통해 행사용 코인을 충전하고, 각 체험 부스에서 QR 코드로 실험비를 결제하는 **블록체인 기반 결제 시스템**의 개발을 성공적으로 완료하였습니다.

현재 로컬 개발 서버가 [http://localhost:5173/](http://localhost:5173/) 주소로 작동 중이며, 코드가 성공적으로 빌드되는 것을 검증하였습니다.

---

## 주요 구현 기능

### 1. 뼈대 블록체인 엔진 (`src/blockchain.ts`)
- **암호학적 해시**: 자체 구현한 동기식 SHA-256 함수를 탑재하여 지연 없이 즉각적인 장부 연산과 UI 업데이트가 가능합니다.
- **트랜잭션**: 개인 키/공개 키 기반의 디지털 서명을 모사하여 관람객이 서명(비밀번호 입력)해야만 거래가 성사되도록 구현했습니다.
- **작업 증명 (PoW)**: 해시 난이도(Difficulty)에 맞춰 채굴(Mining)을 진행하여 실시간 채굴 애니메이션을 연출합니다.
- **무결성 검사**: 블록 해시 값과 이전 블록 해시 링크를 추적하여 임의의 데이터 변조가 감지되면 즉시 에러를 반환합니다.
- **동적 코인 재발행 및 만료 검증**:
  - `reinitialize(coinName, expiryDate, totalSupply)` 메서드를 제공하여 제네시스 블록의 공급 정보를 재생성합니다.
  - `isExpired()`를 통해 현재 시간이 설정된 만료일보다 지난 경우, 트랜잭션 전송(`addTransaction`) 시 에러를 던져 결제 및 충전이 차단되도록 설계했습니다.

### 2. 고품격 사용자 경험 (`src/index.css`)
- **Cyber Space 테마**: 어두운 네이비 배경(`0b0f19`)과 네온 블루, 에메랄드, 퍼플 포인트 그라디언트를 사용했습니다.
- **Glassmorphism**: 브라우저 백그라운드 필터(`backdrop-filter: blur(20px)`)와 극도로 얇은 테두리 반사 효과를 통해 고급스러운 질감을 표현했습니다.
- **마이크로 애니메이션**: 채굴 진행률에 따른 회전 애니메이션, 위변조 감지 시의 경고 펄스 효과, 그리고 성공적인 결제 시 터지는 축하 컨페티 효과를 포함하고 있습니다.

### 3. 관람객 지갑 모드
- **자동 주소 생성**: 관람객 이름에 매핑되는 공개 주소(`0xVisitor_...`) 및 비공개 키(`pv_...`)가 자동으로 생성됩니다.
- **가상 카드 결제 (PG 모사)**: 카드사 선택 및 모의 카드 정보를 입력해 원하는 금액을 충전하면 `CARD_PAYMENT` 주소로부터 관람객 주소로 설정된 커스텀 코인이 충전되는 거래가 발행됩니다.
- **만료일 안내 및 제어**: 지갑 카드에 만료일 정보가 실시간 표기되며, 만료 시 붉은색 경고 띠와 함께 결제 관련 버튼들이 비활성화됩니다.

### 4. 체험 부스 모드
- **부스 등록**: 신규 체험 부스를 동적으로 등록하고 목록에서 개별 부스 매출을 확인합니다.
- **시각적 QR 코드**: 전달값에 맞게 격자 모양이 변하는 **자체 제작 SVG QR 코드**와 센터 코인 아이콘이 결제용으로 렌더링됩니다.
- **결제 상태 대기**: 대기 중, 결제 승인 중(0/1 승인), 결제 완료(1+ 승인) 상태가 실시간으로 변하며 기기 작동 인터랙션을 지원합니다.

### 5. 신규 코인 발행 및 어드민 뷰 (추가 구현)
- **코인 파라미터 셋업**: 관리자 탭에서 **코인 이름**, **유효기간(날짜)**, **최초 발행량**을 입력하고 `신규 코인 발행`을 누르면, 블록체인이 즉시 리셋되며 해당 정보가 담긴 Genesis Block이 런칭됩니다.
- **유동적 명칭 반영**: 코인 이름 설정에 따라 전체 어플리케이션 UI의 수량 단위 표기(예: `SciBlockChain`, `BioToken` 등)가 즉각 연동되어 변환됩니다.
- **해킹(위변조) 데모**: 블록 탐색기에서 임의로 원장 값을 조작해 보안 경고가 발생하는 연쇄 반응을 실습할 수 있습니다.

## 시나리오 테스트 가이드

1. **신규 코인 발행**:
   - [코인 발행 & 어드민] 탭으로 이동합니다.
   - 코인 이름을 `BioToken`으로 지정하고, 유효기간을 내일 날짜로 설정한 후 발행량을 `5,000,000`으로 하여 `신규 코인 발행`을 실행합니다.
   - 전체 UI가 `BioToken` 단위로 리프레시되는 것을 확인합니다.
2. **지갑 충전**: [관람객 지갑] 탭에서 카드 충전을 진행해 잔액을 획득합니다.
3. **부스 결제**: 부스 결제를 수행하고 채굴 및 승인 흐름을 모니터링합니다.
4. **만료 시나리오 테스트**:
   - 어드민 설정에서 유효기간을 어제 날짜(과거)로 수정하여 코인을 재발행합니다.
   - 관람객 지갑 탭에서 **경고: 코인 유효기간 만료** 메세지가 점멸하고 모든 충전/송금 단추가 잠기며, 강제로 전송을 시도해도 블록체인 단에서 에러가 거부되는 것을 확인합니다.
