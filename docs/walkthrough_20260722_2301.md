- [ErrorBoundary.tsx](file:///c:/Work/AntiGravity/SciBlockChain_20260722/src/components/ErrorBoundary.tsx): [main.tsx](file:///c:/Work/AntiGravity/SciBlockChain_20260722/src/main.tsx)에 연동되어 예기치 못한 데이터 오류 시 앱이 무너지지 않도록 안전 복구 UI 제공.
